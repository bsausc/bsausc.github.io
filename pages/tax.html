HowTo: Tax Filing

Federal Tax Return
Federal tax return form can be generated very easily by using cintax online software.
Procedure:
1. International Student Service(ISS) at USC sends cintax password to all international students. If you have not received it, contact ISS for cintax password.
2. Make sure you have following documents ready.

  • Passport
  • I-20 (if F-status) or your DS-2019 (if J-status)
  • Social Security Number
  • Address Information ( both current U.S. address and foreign home address)
  • U.S. entry and exit dates for current and all past visits to the U.S.
  • Academic Institution or Host Sponsor Information. When asked for
    • this information, just enter the ISS office address:

    University of South Carolina, International Student Services,
    901 Sumter St., 123 Byrnes Building,
    Columbia, SC 29208
    Telephone number: (803) 777-7461.
  • Any official tax forms which may include W-2, 1042-S, 1099.
  • A copy of previous year's federal income tax return (form 1040NR or form 1040NR-EZ). It is not necessary of you are filing for the first time
3. Go to www.cintax.us
4. Use the password given by ISS to create an account
5. Follow the instructions to complete the online forms
6. Download and print the tax return file generated by cintax
7. Follow the printed instructions to attach W2 and 1042-S
8. Sign your 1040NR-EZ
9. Make a copy of the signed documents as you will need to attach it with state tax return
10. Mail the tax return to IRS

State Tax Return
1. Goto the following link
http://www.sctax.org/Forms+and+Instructions/
2. Download and print
  • Individual Income Tax Return (SC1040)
  • Nonresident Schedule (Schedule NR)
3. Download "South Carolina Tax Tables (SC1040 Tax Tables)".
4. Use the following two references to fill out SC1040 and ScheduleNR.
Please note that the following references correspond to indivisual tax return and income is assumed to be only from TA/RA.
¶ SC1040
¶ ScheduleNR

* If you have spouse, you can file jointly but your spouse must have SSN or IITN
* If you have child, you should fill out dependent list as you may be eligible for additional tax rebate.
5. Sign the completed form
6. Attach W2, 1042-S, Copy of 'Federal Tax Return'
7. Mail the state tax return to appropriate address
